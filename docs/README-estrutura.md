# Estrutura oficial (versão única)

Projeto oficial:
- `index.html`
- `css/`
- `js/`

Organização interna de scripts:
- `js/core/` -> contratos, validação, estado, utilitários, UI state
- `js/features/` -> app, loja, closet, minigames, render da mascote
- `js/audio/` -> bus de áudio por canal

- `ABRIR_JOGO.bat` -> launcher único oficial
